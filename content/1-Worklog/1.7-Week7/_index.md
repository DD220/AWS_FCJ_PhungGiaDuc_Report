### Week 7 Objectives:

* Organize AWS resources using Tags and Resource Groups.
* Apply AWS Systems Manager for automation and operations.
* Manage resource access through IAM policies.
* Practice system monitoring with Amazon CloudWatch and Grafana.
* Use Session Manager to securely access servers.
